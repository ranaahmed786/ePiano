My Piano Program

Description:

My Piano is a simple piano application built using SFML (Simple and Fast Multimedia Library) in C++. It allows users to play piano notes using both the computer keyboard and mouse.

Features:

	1)Play piano notes using the computer keyboard or mouse clicks.
	2)Visual representation of piano keys on the screen.
	3)Supports both white and black piano keys.
	4)Provides sound feedback for each key press.
Installation;
	Clone this repository to your local machine.
	Make sure you have SFML installed on your system. You can download it from SFML official website.
	Compile the source code using a C++ compiler with SFML linked.
	you can view the installation method on SFML website or from a youtube videos

RUN:

you can run the program by opening the epiano-030-027.sln file and build the solution.
